RNA-Seq Power Tools

This site hosts interactive tools to help you design robust RNA-Seq experiments. Version 2.4 introduces even more realistic planning capabilities based on isoform-level read depth and DESeq2's statistical model.

Launch RNA-Seq Power Calculator v2.4

New in v2.4:

  • Automatically estimate expression (μ) using total reads and number of isoforms
  • Built on DESeq2's Negative Binomial model for count data
  • Plan experiments based on log2FC, dispersion, FDR, and BH rank
  • Evaluate statistical power and estimate minimal replicates per condition
  • Live plots of power vs μ, dispersion, log2FC, and replicates
  • Mobile-friendly, dark-themed interface with legend and formulas included
