Welcome to StatPowerLab

StatPowerLab is a lightweight, browser-based toolkit for designing experiments and estimating statistical power. Whether you're analyzing gene and isoform expression with RNA-Seq technologies, this tool helps you understand how many samples and reads you really need.

Included tools:

  • RNA-Seq Sample Size Estimator
  • Interactive graphs, real-time alerts, and variable legend
