Gene Exon Painter is an interactive tool for mapping exons from multiple transcripts onto a genomic sequence. It helps you:

  • Understand alternative splicing
  • Detect overlapping or shared exons
  • Visualize transcript structures in context
  • Teach gene structure in an intuitive way

ðŸ“˜ How to Use Gene Exon Painter

1. Prepare your files

  • Genomic sequence: file in .fasta, .fa, or .txt
  • Exon FASTA files: headers formatted like >TranscriptID:ExonLabel

2. Load the data

  1. Click Genomic sequence to load your file
  2. Then upload exon files via Exon FASTA files

3. Explore the visualization

  • Select transcript and exon
  • Click an exon to highlight it on sequence
  • Click multiple to show overlaps (with gradients)
  • Click sequence to view exon details

4. Exon Details Panel

  • Shows sequence and genomic position
  • Identical exons in other transcripts
  • Overlapping regions
