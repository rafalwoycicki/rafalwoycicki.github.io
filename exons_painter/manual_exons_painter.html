Gene Exon Painter is an interactive tool for mapping exons from multiple transcripts onto a genomic sequence. It helps you:

  • Understand alternative splicing,
  • Detect overlapping or shared exons,
  • Visualize transcript structures in context,
  • Teach gene structure in an intuitive way.

ðŸ“˜ How to Use Gene Exon Painter

1. Prepare your files

  • Genomic sequence: file in .fasta, .fa, or .txt format.
  • Exon FASTA files: one or more .fasta files. Each header should be in the format >TranscriptID:ExonLabel, e.g. >ENST00000619216:exon_1.

2. Load the data

  1. Click the Genomic sequence input and select your genomic file.
  2. Then use the Exon FASTA files input to load exon files.

3. Explore the visualization

  • Select a transcript to list its exons.
  • Click any exon to highlight it on the sequence.
  • You can activate multiple exons â€“ overlapping regions will be shown with gradient colors.
  • Click a highlighted nucleotide to jump to the corresponding exon.

4. Exon Details Panel

When an exon is selected, you will see:

  • Its sequence and genomic coordinates,
  • Other exons with identical sequence,
  • Overlaps with exons from other transcripts.
